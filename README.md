A modern, responsive eCommerce web application built with React and Vite. This project allows users to browse products, add items to a cart, proceed to checkout, view order history, and track shipments in real-time.

Features
Product Browsing: Display a grid of products with images, ratings, prices, and quantity selectors.
Cart Management: Add products to cart, update quantities, and view cart totals.
Checkout Process: Review order summary, select delivery options, and place orders.
Order History: View past orders with details and totals.
Order Tracking: Track the progress of orders with a visual progress bar showing preparation, shipping, and delivery status.
Search Functionality: Search for products by keywords.
Responsive Design: Optimized for desktop, tablet, and mobile devices.
Real-time Updates: Cart and order data updates dynamically via API calls.
Tech Stack
Frontend: React 19, React Router 7
Build Tool: Vite
Styling: CSS (with responsive media queries)
HTTP Client: Axios
Date Handling: Day.js
Testing: Vitest, React Testing Library
Linting: ESLint
Backend Proxy: Configured to proxy API requests to http://localhost:3000
